Notable courses:

CPSC 329 - Explorations in Information Security and Privacy

  • Introduced to a broad array of topics involved in information security and privacy
  • Learned about legal and ethical considerations in the InfoSec field

CPSC 331 - Data Structure, Algorithms, and Their Analysis

  • Introduced to correctness and analysis of algorithms
  • Learned about fundamental Data Structures (i.e. arrays, lists, stacks, queues, trees, hash tables, etc...)
  • Learned about algorithms for searching and sorting, and utilized Java to implement some of these algorithms

CPSC 418 - Introduction to Cryptography

  • Learned about the basics of Cryptography
    (i.e. symmetric and public key cryptosystems, one-way trapdoor functions, data integrity, data signitures, etc..)
  • Utilized Python and libraries such as SymPy to complete the programming portions of assignments
    (e.g. secure password based authentication, secure file transfer, prefix collision detector)
  • Used mathematical theory to complete the proof-oriented portions of assignments

CPSC 441 - Computer Networks

  • Learning about the principles and practices involved in modern telecommunications, computer communications, and networks
  • Utilizing C to implement programs such as web-proxies, client-server applications, etc...

MATH 271 - Discrete Mathematics

  • Introduced to proof techniques and abstract mathematical reasoning
  • Learned about mathematical induction, primes, modular arithmetic, probability, discrete random variables, etc...

MATH 367 - University Calculus III

  • Introduced to vector calculus, and a deeper dive into differential calculus in serveral variables
  • Learned about Lagrange multipliers, vector functions, line and surface integrals, Green's theorem, Stoke's theorem, Divergence theorem, etc...

MATH 376 - Differential Equations I

  • Introduced to ordinary differential equations with applications
  • Learned about ODE's, series solutions about regular and singular points, special functions, Laplace transform, etc...
